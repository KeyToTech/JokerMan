# JokerMan
This is a primitive version of slack bot to send jokes

To grab jokes it uses these endpoint:

- https://icanhazdadjoke.com/ 
- http://api.icndb.com/jokes/random

## Dependencies are:
- `slackbots` for slack bot API
- `axios` for rest

## Installation

To install all the packages:

` npm install`

To start a bot:

`npm start`
